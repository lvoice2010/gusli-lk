# КиберГусли — Личный кабинет клиента (прототип)

Прототип клиентского ЛК CyberGusli: отчётный дашборд по работе **голосового**
AI-ассистента. Стек — Next.js 14 (App Router) + TypeScript + Tailwind.
Брендинг — тёмная тема КиберГусли (фиолетовый → синий → циан, мотив звуковой волны).

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000 — произойдёт редирект на `/dashboard`.

## Структура ЛК

Боковое меню (по образцу O'LINE): **Главная · Ассистенты**, группа **Управление**
(Фин. документы · Отчёты · Отправить сообщение) и выделенная цветом **Витрина услуг**.

Внизу сайдбара — переключатель **тёмная/светлая тема** (тумблер, сохраняется в
localStorage) и профиль клиента (имя, фамилия, компания). Оплата **по факту** (без пакета
минут). Дашборд ассистента открывается из списка «Ассистенты» / карточки на «Главной».

Вся работа по ассистенту собрана в одном месте (как на портале O'LINE): статистика +
диалоги (звонки) + аналитика + используемая база знаний — через вкладки, без отдельного
раздела «Звонки».

### Дашборд ассистента (`/dashboard`) — ядро прототипа
Заголовок услуги (статус, оплата по факту, менеджер, «Скачать отчёт») и 4 вкладки:

**Отчёты** — единый **переключатель периода** (Сегодня / Вчера / Неделя / Месяц /
Период) со сравнением к **сопоставимому** периоду и виджеты:
- **KPI-сводка**: обращения, закрыто ИИ, эскалации + %, средняя длительность
  (до секунд), стоимость.
- **ИИ vs оператор** — доля закрытых ИИ против переданных, с разбивкой по причинам.
- **Классификация** (целевые / повторные / купившие / нецелевые).
- **Тематика** обращений + динамика.
- **Воронка** с тултипами-определениями на каждом шаге.
- **ROI / экономия** — сэкономленные минуты оператора и деньги по ставке из настроек.
- **Статус интеграций** — только подключённые (Restoplace); остальные с честным
  пустым состоянием.

**Аналитика** — динамика обращений (линия) + почасовая карта нагрузки.

**Диалоги** — журнал звонков: колонки **Тема** и **Эскалация** (да/нет + причина),
понятные статусы, кнопка **«Открыть»**, кликабельная строка, карточка звонка с
расшифровкой, фильтр **тест / боевые / все**.

**База знаний** — используемая база знаний ассистента (просмотр; изменения — на стороне
КиберГусли).

Кнопка **«Скачать отчёт (PDF)»** — печать чистой версии (`@media print`).

## Данные

Все цифры — демонстрационные (см. `lib/mock.ts`), согласованы между виджетами.
Где данных нет (неподключённые интеграции, база знаний) — честные пустые состояния,
цифры не выдумываются.

## Заметки по реализации

- Графики — собственные на SVG (`components/charts/*`), без тяжёлых зависимостей.
- Только **голосовые** ассистенты (текстовых каналов нет).
- Только **входящие** звонки (исходящие появятся позже).
- Вся статистика ассистента — внутри его страницы; отдельного раздела «Звонки» нет.
- Единый фиксированный тариф — раздела «Тарифы» нет.
- В текущем (интеграторском) кабинете раздел «Глоссарий» не выводим.
